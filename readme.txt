Generates report of the status of subscriptions

== Description ==

It generates reports of the status of the subscriptions in excel, sending by email, this is linked its use with the Subscription Payu Latam plugin

== Installation ==

1. Download the plugin
2. Enter the administrator of your WordPress.
3. Enter Plugins / Add-New / Upload-Plugin.
4. Find the plugin downloaded on your computer and upload it like any other file.
5. After installing the .zip you can see it in the list of installed plugins, you can activate or deactivate it.

== Frequently Asked Questions ==

= How to generate a report? =

As simple as indicating the start date, end date and the status of the subscriptions
